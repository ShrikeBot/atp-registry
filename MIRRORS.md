# Known ATP Mirrors

List of known ATP registry mirrors. Anyone can run a mirror.

## Official

| Mirror | Operator | URL |
|--------|----------|-----|
| ShrikeBot/atp-registry | @ShrikeBot | https://github.com/ShrikeBot/atp-registry |

## Running Your Own Mirror

1. Fork this repository
2. Run the [atp-mirror](https://github.com/ShrikeBot/atp-mirror) bot
3. Point it at your fork
4. Submit PR to add your mirror to this list

## Conflict Resolution

All mirrors read from Bitcoin. If mirrors disagree, Bitcoin is correct.
